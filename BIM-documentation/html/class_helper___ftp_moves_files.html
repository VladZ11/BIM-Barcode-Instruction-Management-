Detailed Description

Helper class for FTP file operations in the BIM system.

Author
Vladyslav Zhyokin

This class handles FTP operations for image files associated with notes in the Barcode Instruction Management (BIM) system. It provides functionality for connecting to an FTP server, uploading files, deleting files, and properly closing connections.
