Detailed Description

Class responsible for rendering main application pages.

Author
Vladyslav Zhyokin

This class manages the display and rendering of all templates related to the main interface of the Barcode Instruction Management (BIM) system. It provides methods for displaying the home page, search results, and instruction data views.
