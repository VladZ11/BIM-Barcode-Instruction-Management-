Detailed Description

SQL creator class for order management in the BIM system.

Author
Vladyslav Zhyokin // Dariusz Bakiewicz

This class handles database interactions related to component orders, reservations, and warehouse location management. It provides functionality for adding components to orders, managing reservations, updating component locations, and handling transactions. It interacts primarily with the 'ewa_orders', 'ewa_warehouse', and 'ewa_reservations' database tables.
