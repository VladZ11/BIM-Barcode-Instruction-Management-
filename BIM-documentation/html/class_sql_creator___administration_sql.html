Detailed Description

SQL creator class for administration operations in the BIM system.

Author
Vladyslav Zhyokin

This class manages database interactions for the administration module of the Barcode Instruction Management (BIM) system. It provides methods for managing notes, images, program configurations, and instruction data. The class handles CRUD operations (Create, Read, Update, Delete) for various entities including general information records, header data, print programs, dock programs, and notes.
