Detailed Description

Class responsible for rendering orders-related pages.

Author
Vladyslav Zhyokin

This class manages the display and rendering of all templates related to the orders section of the Barcode Instruction Management (BIM) system. It provides methods for handling reservations, order management, and order history displays.
