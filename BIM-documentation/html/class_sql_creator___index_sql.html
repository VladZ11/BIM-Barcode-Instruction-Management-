Detailed Description

SQL creator class for the BIM system index functionality.

Author
Vladyslav Zhyokin

This class manages database interactions for the main index functionality of the Barcode Instruction Management (BIM) system. It connects to the QDM database and provides methods for retrieving barcode data, instruction information, notes, and component details. It also handles queries related to SMD processes including programs, maps, and equipment configurations.
