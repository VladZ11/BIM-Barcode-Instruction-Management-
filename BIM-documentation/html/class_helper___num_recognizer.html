Detailed Description

Helper class for identifying and processing various component number types.

Author
Vladyslav Zhyokin

This class provides functionality for validating and extracting information from different component identification numbers in the manufacturing system, including:

  • WPN (Work Process Number)
  • DID (Direct Identification)
  • SSCC (Serial Shipping Container Code)

It connects to the database to retrieve complete information about components based on their identification numbers, including warehouse location, part properties, supplier information, and more. It also integrates with SAP/REST services for SSCC code validation and additional data retrieval.
