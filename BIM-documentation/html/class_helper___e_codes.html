Detailed Description

Error code handling class for the BIM system.

Author
Vladyslav Zhyokin

This class is responsible for mapping error codes to appropriate messages and error types. It provides a centralized way to handle error messages throughout the application, supporting multiple error scenarios with customizable messages.
